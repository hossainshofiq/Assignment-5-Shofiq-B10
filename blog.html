Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents and manipulates the structure and content of web documents as a tree of objects.

How do you select an element from the DOM?

You can select an element from the DOM using methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName(). These allow you to target elements by ID, class, or CSS selectors.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events from its child elements. It is useful because it improves performance by reducing the number of event listeners and allows dynamic elements to be handled efficiently.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, you can use methods like setAttribute(), getAttribute(), and removeAttribute(). For styles, you can modify the style property directly, e.g., element.style.color = "red";, or by adding/removing CSS classes using classList.add() or classList.remove().
